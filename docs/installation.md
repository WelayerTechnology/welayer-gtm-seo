# Installation

## Community Gallery
1. Open GTM.
2. Templates.
3. Community Template Gallery.
4. Search **WeLayer Tracking**.
5. Review permissions.
6. Install.

## Required variables

- `wlyr.token` (Constant)
- `wlyr.gtm.container` (Container ID)

## Configure the tag

- Bearer Token
- GTM Container ID
- Legal Version
- Debug Mode

## Triggers

Create Custom Event triggers only for supported business events.

Do not trigger the tag for:
- gtm.js
- gtm.dom
- gtm.load

Validate using GTM Preview before publishing.
